bare node

Bare nodes are mesh nodes that contain other mesh nodes. Their identifier

Function

  • namespace extenders and perhaps organizational containers

Mandatory Components: _meta/ + _node-handle/ Optional Components: meshnode config flows, Documentation Resource

They are physically represented by node folder.

Purpose

  • scaffolding, grouping, deferred semantics
  • a secondary, optional function is as "semantic contextualizers", but bare nodes don't have any definitive Referent of their own.
