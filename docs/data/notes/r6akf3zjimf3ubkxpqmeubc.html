snapshot

A snapshot dataset generated "on weave"

The Snapshot folder is named with the Weave Label plus the Flow Version

Disambiguation

  • a snapshot is an addressable resource;
  • it is differentiated from the concept of a unversioned flow facet in that a versioned flow is a flow that has never had a version.
