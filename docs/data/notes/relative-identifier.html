relative identifier

Intramesh relative identifiers (or relative identifiers for short) are relative IRIs that do not start with a slash (/), i.e., they are relative-path relative IRIs that correspond to a mesh location and denote an internal or external referent..

Relative identifiers are always resolved relative to the distribution file that contains them.

See Reference Iri Choices for discussion and examples.
