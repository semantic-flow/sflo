reference flow

  • one of the semantic flow elements
  • used for attaching data to a reference node
  • logically, it's a versioned component, so contains a v-series component facet
  • physically represented as a reference flow folder
  • not a MeshNode, so doesn't have its own metadata flow
    • metadata is kept in its parent node's metadata flow
