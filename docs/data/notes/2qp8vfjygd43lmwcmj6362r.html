Working Distribution

Unlike most other flow snapshot, the next snapshot is user-facing (directly modifiable), and so there shouldn't be any Sibling Distribution. Just the one distribution that may be updated, and if multiple-syntax weaving is turned-on, is the source for the siblings.

The CLI should have functionality for converting the working distribution between formats, in case you want to change the syntax you're using.
