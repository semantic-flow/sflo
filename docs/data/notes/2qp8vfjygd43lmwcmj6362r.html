working distribution

Unlike most other FlowShot, the working snapshot is user-facing (directly modifiable), and so there shouldn't be any Sibling Distribution. Just the one distribution that may be updated, and if multiple-syntax weaving is turned-on, is the source for the siblings.

The sflo CLI should have functionality for converting the working distribution between formats, in case you want to change the syntax you're using.
