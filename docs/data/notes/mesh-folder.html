mesh folder

Semantic meshes are represented in filesystems as a collection of folders and files. Mesh folders correspond to naming resource.

With the exception of assets tree folder, mesh folders should only contain other mesh resources, i.e., other mesh folders and mesh file, no "arbitrary files."
