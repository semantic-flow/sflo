Semantic Site

Semantic Flow sites provide two main services:

  • hosting persistent, namespaced URLs for use in the semantic web
  • hosting versioned RDF datasets, and their histories

Publishing

  • For sites exposed by Github/Gitlab Pages functionality, pushing a mesh repo effectively publishes it.
