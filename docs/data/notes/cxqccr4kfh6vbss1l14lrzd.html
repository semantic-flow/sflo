Snapshot folder

  • the name of snapshot folders, and therefore part of their IRIs.
  • format: YYYY-MM-DD_HHMM_SS_vN (e.g., 2025-11-24_0142_07_v1)
  • combines the Weave Label (YYYY-MM-DD_HHMM_SS) with the sequence number from Flow Version (_vN)
