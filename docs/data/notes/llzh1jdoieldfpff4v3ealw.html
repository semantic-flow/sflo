root node

The node at the top of a mesh hierarchy may be referred to as the root node.

Every other mesh resources in a mesh "lives under" the root node.

For pure mesh repos, the repository's name is used as root node's identifier.

For embedded mesh, the root node's folder name is its identifier.

A root node is not treated or represented any differently than any other mesh node, and it is not differentiated in metadata. So any node may become a root node simply by copying it somewhere that's not already a mesh.
