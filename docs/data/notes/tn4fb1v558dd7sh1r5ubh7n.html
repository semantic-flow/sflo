mesh data-series node

A mesh dataset-series node is not generally distinguished from data nodes in terms of file structure, but their metadata should identify them as related-topics.dcat.dataset-series (Private) and identify the datasets in their series.

The contained datasets don't need to physically live inside the series' folder, but that can be an intuitive design pattern.

versioned component are not mesh dataset-series themselves, but they contain v-series component facet which are. (And version snapshot that are not.)
