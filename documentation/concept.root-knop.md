The knop at the top of a mesh hierarchy may be referred to as the root knop. 

Every other [[mesh-resource]] in a mesh "lives under" the root knop.

For pure [[concept.single-mesh-repo]]s, the repository's name is used as root knop's identifier. 

For [[concept.mesh.embedded]], the root knop's folder name is its identifier.

A root knop is not treated or represented any differently than any other [[mesh-resource.knop]], and it is not differentiated in metadata. So any knop may become a root knop simply by copying it somewhere that's not already a mesh.
