A chunk of a path, i.e., usually between the slashes. Because IRI paths (below the base) reflect filesystem paths, there is a direct correspondence to filesystem directory (aka folder) names. But "path segment" is the recommended term as it can apply equally to IRI segments and filesystem path segments.
