The **working folder** serves as a draft workspace for ongoing changes to a knop's [[mesh-resource.component.flow]]'s [[mesh-resource.component.distribution]].

During the [[concept.weave-process]], modified [[mesh-resource.component.distribution.working]] are copied into memory, at which point they remain ready for further changes.

This allows almost-continuous availability for editing and continuous access for users of the stable slices.

These folders should only contain a single "[[mesh-resource.component.distribution.working]] file" plus an optional [[mesh-resource.component.documentation-resource.resource-page]]
