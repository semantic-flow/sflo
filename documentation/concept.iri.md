On the semantic web, resources (things you might want to talk about with RDF statements) are identified by Internationalized Resource Identifiers (IRIs).

IRIs are basically URLS, except they don't necessarily LOCATE a resource, i.e., if you put them in a web browser, they don't necessarily return content.

But with [[Semantic Flow sites|concept.semantic-flow-site]], all [[NonArtifact]] IRIs return a web page. That is:

- they identify a referent, AND
  - if it's [[facet.resource.content]], they locate the content
  - if it's [[facet.resource.naming]], they locate a resource page about the referent. 


## Kinds of IRIs

- Absolute IRIs
- Relative IRIs
  - Relative-Path Relative IRIs
  - Absolute-Path Relative IRIs 

See [[faq.reference-iri-choices]] for a discussion of IRI types.
